# robisoft-legacy-system-knowledge-app
Web Application to provide essential information about legacy programs and applications (COBOL, PL1, etc.)

The following five questions are asked. (Alternatively, the user may ask a custom question).

1.  "What does this program do?"
2.  "What business rules exist?"
3.  "What are the inputs and outputs expected?",
4.  "What modernization opportunities exist?",
5.  "What dependencies are visible?",
